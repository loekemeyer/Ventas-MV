# Ventas MV

Aplicación web para cargar y consultar ventas de indumentaria a medida.

## Características

- **Módulo Cargar**: formulario con cliente, teléfono, prenda, material (Oveja/Vaca/Gamuza), color, talle (XS–XL o A medida), importe y forma de pago (Efectivo/Transferencia/Otro).
- **Módulo Registro**: listado con búsqueda, filtros por fecha, estadísticas y exportación a CSV.
- Optimizada para iPhone (responsive, sin zoom automático, safe-area-insets).
- Funciona sin conexión (localStorage).
- Accesos directos a WhatsApp y llamada desde cada venta.

## Uso

Abrir `ventas.html` en un navegador. No requiere servidor ni instalación.
